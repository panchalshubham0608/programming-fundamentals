Goal: Student should know `Hello World` program!

---

#### Write a program to print "Hello World" (without quotes)
```
Hello World
```

#### C Solution
```c
#include <stdio.h>
int main(){
    printf("Hello World");
    return 0;
}
```


#### C++ Solution
```cpp
#include <iostream>
using namespace std;
int main(){
    cout <<"Hello World";
    return 0;
}
```
